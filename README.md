Drone-Diaster-Communication is a drone-based emergency communication system for disaster zones. It helps restore basic network access, detect possible survivors, and support rescue teams during field work.

The app is built for quick use in urgent situations. It gives a simple way to manage aerial base station support, track communication coverage, and view rescue-related data in one place.
